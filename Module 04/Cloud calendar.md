* Cloud collaboration uses the internet to store files and tools online.
* Calendar collaboration is typically done via platforms such as **Google Calendar, Microsoft Outlook Calendar, Apple iCloud Calendar**, etc.

  * **Share calendars** with team members.
  * **Set reminders and deadlines** that notify everyone.
  * **See availability** of collaborators to schedule meetings efficiently.

**3. Examples of Tools**

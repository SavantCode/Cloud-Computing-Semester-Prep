# Business Models around Cloud ☁️

## 1. What is a Business Model in Cloud Computing?

A **business model** explains **how a cloud company provides services and earns money from them**.

In cloud computing, companies provide **IT resources (servers, storage, software, platforms)** through the internet and **charge customers based on usage**.

Instead of buying expensive infrastructure, businesses **rent services from cloud providers**.

Example cloud companies include:

* Amazon Web Services
* Microsoft Azure
* Google Cloud

These companies operate large **data centers** and provide services to customers globally.

# Main Business Models in Cloud Computing

Cloud business models are mainly based on **how services are sold and delivered**.

The major models are:

### Explanation

In this model, customers **pay only for the resources they use**.

This is similar to **electricity billing**.

If you use more resources → pay more
If you use less → pay less

### Example

A company uses:

* 100 GB storage
* 2 virtual servers

They pay only for those resources.

Providers using this model include:

* Amazon Web Services
* Google Cloud

### Advantages

* Cost efficient
* Flexible usage
* No upfront investment

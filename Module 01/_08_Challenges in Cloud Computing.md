# 1. What are Challenges in Cloud Computing?

Cloud computing provides many advantages, but organizations may face **technical, security, and management challenges** when using cloud services.

These challenges mainly include:

1. Migration Challenges
2. Integration Challenges
3. Proprietary vs Open Source Issues

---

# 1. Migration Challenges

## What is Migration in Cloud Computing?

**Cloud migration** means **moving applications, data, and IT infrastructure from local systems (on-premise) to the cloud**.

Example:
A company moves its website from its **local data center to Amazon Web Services cloud servers.**

---

## Migration Challenges

### 1. Data Transfer Difficulty

Moving **large amounts of data to the cloud** may take a lot of time.

Example:
A company with **10 TB of data** migrating to cloud servers.

---

### 2. Application Compatibility

Some applications designed for **local servers may not work properly in the cloud**.

They may require modification.

---

### 3. Downtime Risk

During migration, systems may become temporarily unavailable.

This can affect business operations.

---

### 4. Security Concerns

Sensitive data must be transferred securely to prevent **data breaches**.

---

# 2. Integration Challenges

## What is Integration?

**Cloud integration** means **connecting cloud systems with existing on-premise systems and other cloud services**.

Many organizations use a **hybrid environment**, where some systems run locally and others run in the cloud.

* cloud storage system
* online application

These systems must communicate with each other.

---

## Integration Challenges

### 1. System Compatibility

Different systems may use **different technologies and data formats**.

Integrating them can be difficult.

---

### 2. Data Synchronization

Data stored in local systems and cloud systems must remain **consistent and updated**.

---

### 3. Network Latency

Communication between local systems and cloud servers may introduce **delays**.

---

### 4. Security Issues

Data moving between systems must be **protected from unauthorized access**.

# 3. Proprietary vs Open Source Issues

Organizations must choose between:

* **Proprietary cloud solutions**
* **Open-source cloud platforms**

Each has advantages and disadvantages.

---

## Proprietary Cloud Platforms

These are **commercial cloud solutions owned by companies**.

Examples include:

* Amazon Web Services
* Microsoft Azure
* Google Cloud

### Advantages

* reliable infrastructure
* strong support
* advanced features

### Disadvantages

* vendor lock-in
* higher cost
* limited customization

---

## Open Source Cloud Platforms

These are **free cloud platforms whose source code is publicly available**.

Examples include:

* OpenStack
* Eucalyptus
* OpenNebula

### Advantages

* free or low cost
* customizable
* no vendor dependency

### Disadvantages

* requires technical expertise
* limited official support

---

## Proprietary vs Open Source Comparison

| Feature       | Proprietary Cloud    | Open Source Cloud |
| ------------- | -------------------- | ----------------- |
| Ownership     | Private company      | Community         |
| Cost          | Paid                 | Mostly free       |
| Customization | Limited              | High              |
| Support       | Professional support | Community support |

---

# Real-Life Example

A company deciding cloud strategy:

Option 1
Use **Amazon Web Services** (proprietary cloud)

Option 2
Build private cloud using **OpenNebula** (open-source cloud)

The decision depends on **budget, expertise, and security requirements**.

---

# Short Exam Conclusion

Cloud computing adoption involves several challenges such as **migration difficulties, integration with existing systems, and choosing between proprietary and open-source platforms**. Organizations must carefully evaluate these factors to ensure successful cloud deployment.

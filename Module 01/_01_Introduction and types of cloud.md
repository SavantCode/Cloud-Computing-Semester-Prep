# 1. Cloud Introduction ☁️

Cloud computing is a technology that allows users to **store data, run applications, and access computing resources over the internet instead of using a local computer or server**.

Earlier, companies had to **buy expensive servers, storage devices, and networking equipment** to run applications. These systems required **maintenance, electricity, space, and technical staff**.

Cloud computing solved this problem by allowing users to **rent computing resources from cloud providers through the internet**.

Instead of owning hardware, users can access services **on demand** from cloud providers like:

* Amazon Web Services
* Microsoft Azure
* Google Cloud

These providers maintain large **data centers** and provide computing services globally.

### Example

When you store files on **Google Drive**, watch movies on **Netflix**, or use **Gmail**, you are using **cloud computing**.

---

# 2. Cloud Computing Fundamentals

Cloud computing fundamentals are the **basic principles that explain how cloud systems work**.

### Key fundamentals

#### 1. On-demand self service

Users can **automatically get computing resources** like storage or servers whenever they need without contacting the provider.

Example: Creating a virtual server in minutes.

---

#### 2. Broad network access

Cloud services are available **over the internet** and can be accessed from:

* laptops
* smartphones
* tablets
* desktops

---

#### 3. Resource pooling

Cloud providers **pool their computing resources** and serve multiple customers using the same infrastructure.

This is called **multi-tenancy**.

Example: One large data center serving thousands of companies.

---

#### 4. Rapid elasticity

Resources can be **quickly increased or decreased** based on demand.

Example:

* During **festival sales**, an e-commerce website increases servers.
* After the sale, it reduces them.

---

#### 5. Measured service (Pay-as-you-go)

Users **pay only for what they use**.

Example:
If you use **10GB storage**, you pay only for that amount.

---

# 3. Cloud Computing Definition

**Definition:**

> Cloud computing is a model that enables convenient, on-demand network access to a shared pool of configurable computing resources such as networks, servers, storage, applications, and services that can be rapidly provisioned and released with minimal management effort.

In simple words:

Cloud computing means **using computing services through the internet instead of owning physical hardware**.
